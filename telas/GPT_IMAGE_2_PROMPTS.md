# GPT Image 2 Prompt Book

## Global direction
Use a clean, premium, operational UI for SendSprint. The visual reference is a blend of Codex blue accents, Claude Desktop cleanliness, enterprise control-plane density, and modern AI operator workspaces. Prefer light backgrounds, blue-driven hierarchy, soft panels, strong spacing, monospace touches for logs, clean typography, and deliberate information architecture. Avoid generic dashboards, purple gradients, noisy neumorphism, or game-like cards.

## Shared visual system
- brand: SendSprint
- tone: calm, precise, technical, executive-ready
- palette: cloud white, steel blue, codex blue, slate text, subtle aqua success, coral danger
- style: crisp cards, thin borders, generous whitespace, soft shadows, precise tables, kanban columns, side panels, command surfaces, modular enterprise navigation
- typography: premium grotesk + monospace support for logs and branches
- constraints: no purple bias, no dark-mode-only output, no childish iconography
